📚 Topics Covered
🧩 OSI Model

The OSI (Open Systems Interconnection) model has 7 layers:

Physical
Data Link
Network
Transport
Session
Presentation
Application

👉 Helps in understanding how data travels across a network.

🌍 TCP/IP Model

A simplified version of OSI with 4 layers:

Application
Transport
Internet
Network Access

👉 This is the real-world networking model used on the internet.

🔢 IP Addressing
IPv4 (e.g., 192.168.1.1)
IPv6 (modern version)
Public vs Private IP

👉 Used to identify devices on a network.

🌐 DNS (Domain Name System)
Converts domain names into IP addresses
Example: google.com → IP address

👉 Makes internet browsing easy.

🔌 Networking Devices
Router
Switch
Firewall

👉 These devices help manage and secure network traffic.
